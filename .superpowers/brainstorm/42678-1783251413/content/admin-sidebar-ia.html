탭별 첫 버전 범위

바로 연결 가능한 탭

  • 대시보드: analytics, action logs, reports/payments summary
  • 사용자: 검색, 상세, 이벤트, 해시태그, 크레딧 원장
  • AI 캐릭터: 목록, 생성, 수정, 상태, 메모리, 게시·생성 액션
  • 콘텐츠/미디어: 미디어 목록, 업로드 확인, 게시물 작성
  • 신고/모더레이션: 신고 목록, 상세, 상태 변경
  • 결제/정산: reconciliation 목록, 결제 상세

나중에 붙일 것

  • 권한/역할 관리: 현재 API 없음
  • 고급 차트: 표와 숫자로 먼저 충분
  • 새 디자인 시스템/프레임워크: dependency-free 유지
